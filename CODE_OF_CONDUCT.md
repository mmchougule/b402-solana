# Code of Conduct

We follow the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The short version:

- Be respectful in code review, issue threads, and Discord/X conversations
  about the project.
- Critique code, not people.
- No harassment, threats, or discrimination of any kind. Privacy projects
  attract trolling — we don't tolerate any of it from any side.
- Disagreements about technical direction are fine and expected; resolve
  them in PRs and design docs, not flame wars.

Reports of conduct violations: email **conduct@b402.ai**. Reports are
treated confidentially. Maintainers can ban offenders from the repository
and associated chat channels.

The full Contributor Covenant text applies. The above is a summary, not a
substitute.
